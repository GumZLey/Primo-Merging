## Setup

### 1. Clone the Repository

```bash
git clone [https://github.com/your-github-username/sorted-merge.git](https://github.com/your-github-username/sorted-merge.git)
cd sorted-merge
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the code

```bash
npx ts-node src/main.ts
```

### 4. Run the test

```bash
npm test
